# Taller Independiente - Pasar los 3 wireframes a Código html y css
En este repositorio esta la tarea Independiente sobre html y css, a través de estos se tiene que hacer los 3 wireframes a html y css, donde queden casi igual.
Cada carpeta (Punto 1, Punto 2 y Punto 3) tiene lo siguiente
- Index.html: Aquí esta la estrcutura del wireframe con sus debidos contenedores para hacer más ameno el trabajo en css.
- estilo.css - Aquí esta el diseño visual y ya organizado para que quede lo más parecido a los wireframes.
